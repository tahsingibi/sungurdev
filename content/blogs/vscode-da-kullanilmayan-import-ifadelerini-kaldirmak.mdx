VSCode'da kullanılmayan import ifadelerini kaldırmak için `.vscode/settings.json` dosyanıza eklemeniz gereken kod;

```javascript/filename=.vscode/settings.json
{
    "editor.codeActionsOnSave": {
        "source.organizeImports": true
    }
}
```
